函数

方法

  • 方法是依附于对象的函数,通过 self 来访问对象中的数据.在 impl 中定义
    • 可以为一个结构其定义多个 impl 块
  • 方法的写法 f.read(buffer)
  • 方法往往跟结构体.枚举.特征 一起使用
#[derive(Debug)]
struct Rectangle {
    width: u32,
    height: u32,
}

impl Rectangle {
    // &self是 self:&Self 的简写
    // self指实例,Self指类型
    fn area(&self) -> u32 {
        self.width * self.height
    }
}

fn main() {
    let rect1 = Rectangle { width: 30, height: 50 };

    println!(
        "The area of the rectangle is {} square pixels.",
        rect1.area()
    );
}
  • self 表示 Rectangle 的所有权转移到该方法中，这种形式用的较少
  • &self 表示该方法对 Rectangle 的不可变借用
  • &mut self 表示可变借用

好处

  • 不用在函数签名中重复书写 self 对应的类型
  • 利于维护和阅读

函数

函数的写法: read(f.buffer)

  • 函数与方法的一个区别就是:函数没有 self,不能用 f.read() 形式调用

定义

函数定义也是语句,语句不返回值,表达式是语句的一部分
(函数调用.宏调用.块作用域都是表达式)

返回值

无返回值的函数,实际会返回一个单元类型 ()

// 相同
fn fizzbuzz(n: u32) -> () {}
fn fizzbuzz(n: u32){}

关联函数

  • 也被称作 静态方法 或者 构造器
impl Rectangle {
    fn new(w: u32, h: u32) -> Rectangle {
        Rectangle { width: w, height: h }
    }
}
  • 定义在 impl 中且没有 self 参数的函数
  • 关联函数只能用双冒号 rect1::new() 的形式来调用
  • 一般用 new 来作为构造器的名称

闭包

捕获周围作用域中的变量的函数,
在临时使用时很方便

let closure_inferred = |i|i+1;
closure_inferred(3);
