泛型

定义

  • 泛型是通过抽象函数用来简化代码,静态分发
  • 用 T,E 等泛型参数来替代参数类型
  • 泛型没有性能开销,代价是损失了编译速度和增大了生成文件的大小
fn add<T>(a:T, b:T) -> T { // fn add<T: std::ops::Add<Output = T>>(a: T, b: T) -> T{
    a + b
}

fn main() {
    println!("add i8: {}", add(2i8, 3i8));
    println!("add i32: {}", add(20, 30));
    println!("add f64: {}", add(1.23, 1.23));
}

显式地指定泛型的类型参数

  • 给泛型参数加约束(bound)
    fn add< /T: std::ops::Add<Output = T>/ >(a: T, b: T) -> T 相当于 where
    fn add<T>(a: T, b: T) -> T /where T: std::ops::Add<Output = T>/ ,

  • 在调用泛型函数时,要加具体类型

      use std::fmt::Display;

  fn create_and_print<T>() where T: From<i32> + Display {
      let a: T = 100.into(); // 创建了类型为 T 的变量 a，它的初始值由 100 转换而来
      println!("a is: {}", a);
  }

  fn main() {
      create_and_print::<i64>();
  }

结构体中使用泛型

  struct Point<T> {
      x: T,
      y: T,
  }

  fn main() {
      let integer = Point { x: 5, y: 10 };
      let float = Point { x: 1.0, y: 4.0 };
  }
  • 泛型参数要提前声明

枚举中使用泛型

  • Option
enum Option<T> {
    Some(T),
    None,
}
  • Result
enum Result<T, E> {
    Ok(T),
    Err(E),
}

方法中使用泛型

struct Point<T> {
    x: T,
    y: T,
}

impl<T> Point<T> {
    fn x(&self) -> &T {
        &self.x
    }
}

fn main() {
    let p = Point { x: 5, y: 10 };

    println!("p.x = {}", p.x());
}
  • 形式为 impl<T> method<T> {}

为具体的泛型类型实现方法

impl Point<f32> {
    fn distance_from_origin(&self) -> f32 {
        (self.x.powi(2) + self.y.powi(2)).sqrt()
    }
}

Point<f32> 类型会有一个方法 distance_from_origin，而其他 T 不是 f32 类型的 Point<T> 实例则没有定义此方法

const 泛型

fn display_array<T: std::fmt::Debug, const N: usize>(arr: [T; N]) {
    println!("{:?}", arr);
}
fn main() {
    let arr: [i32; 3] = [1, 2, 3];
    display_array(arr);

    let arr: [i32; 2] = [1, 2];
    display_array(arr);
}
  • T 是一个基于类型的,实现 Debug 特征的参数
  • N 是一个基于值的,const usize 类型的参数
  • 方便数组的泛型,特别使长度方面

const fn

  • 主要是通过 const fn 源码告诉读者这个值的来源
const fn add(a: usize, b: usize) -> usize {
    a + b
}

const RESULT: usize = add(5, 10);

fn main() {
    println!("The result is: {}", RESULT);
}
  • const fn 允许我们在编译期对函数进行求值
  • 不能将随机数生成器写成 const fn
